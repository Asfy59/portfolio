# Asfand Portfolio Site

Professional portfolio site built with Jekyll + Minimal Mistakes.

## Local development

Prerequisites:
- Ruby and Bundler installed

Run:

```bash
bundle install
bundle exec jekyll serve
```

Open `http://127.0.0.1:4000/asfand/`

## Deploy on GitHub Pages

1. Push this repository to `Asfy59/asfand`.
2. In GitHub, go to **Settings -> Pages**.
3. Set source to **Deploy from a branch**.
4. Choose `main` branch and `/ (root)` folder.
5. Save and wait for deployment.

Site URL:
- `https://asfy59.github.io/asfand/`

## Customize

- Edit site-wide settings in `_config.yml`.
- Edit top navigation in `_data/navigation.yml`.
- Edit content pages in `_pages/`.
- Add resume PDF to `assets/files/Asfand_Resume.pdf`.
