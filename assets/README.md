# Portfolio Assets

Use this structure to keep website content organized.

- `images/profile/`: profile photos
- `images/projects/`: project screenshots, charts, diagrams
- `images/experience/`: work/lab photos
- `images/publications/`: publication figures
- `presentations/`: slide decks (PDF/PPTX)
- `docs/`: reports, papers, technical writeups
- `certificates/`: certificates and attestations

Recommended naming format:
- `topic-context-YYYY-MM.ext`

Example:
- `radar-fusion-demo-2025-02.png`
